# Read-First, Evaluated Writes

Nudge starts with read-only integrations and draft-only suggestions because the product handles sensitive personal memory, calendar context, and relationship context. Write capabilities may be added later, but only behind a Human-in-the-Loop approval pattern and after the system can evaluate whether a proposed action or behavior change is useful, safe, reversible, and grounded in source evidence; research on agentic memory, self-evolving agents, and harness evolution should inform those gates.
