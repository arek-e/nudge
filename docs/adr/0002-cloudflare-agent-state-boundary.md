# Cloudflare Agent State Boundary

Nudge uses Cloudflare Agents SDK and Durable Objects for live per-user agent/session coordination, starting with active conversations. Pending Review Queue waits and Resume Token wakeups are future slices. ADR 0016 supersedes the earlier D1 product-store direction: Convex is the durable source of truth for events, memories, Review Queue items, decisions, evaluations, notes, and workflow outcomes, while Workers Workflows handle long-running multi-step jobs such as note analysis, Digest generation, memory consolidation, and Human-in-the-Loop pause/resume. Deployed Workers, Agents, and Workflows do not bind D1; `packages/db` keeps only the persistence interface and memory adapter.
